# Modem Pool Manager - مدير المودمات

## نظرة عامة
برنامج C# WPF لإدارة مودمات متعددة (حتى 12 مودم) وتنفيذ أوامر USSD بشكل متزامن على جميع المودمات.

## التقنيات المستخدمة
- **اللغة**: C# 12
- **الإطار**: .NET 8.0
- **الواجهة**: WPF (Windows Presentation Foundation)
- **النمط**: MVVM (Model-View-ViewModel)

## هيكل المشروع
```
ModemPoolManager/
├── Models/           # نماذج البيانات
├── Services/         # خدمات إدارة المودمات
├── ViewModels/       # ViewModels للواجهة
├── Converters/       # محولات البيانات
├── MainWindow.xaml   # الواجهة الرئيسية
└── App.xaml          # تعريف التطبيق
```

## الميزات الرئيسية
1. اكتشاف تلقائي لمنافذ ZTE Diagnostics
2. جلب أرقام الهواتف من المودمات
3. تنفيذ أوامر USSD بشكل متزامن
4. واجهة عربية حديثة

## ملاحظة مهمة
هذا البرنامج يعمل فقط على Windows لأن WPF خاص بـ Windows.
يمكنك تحميل الكود وتشغيله على جهاز Windows الخاص بك.
